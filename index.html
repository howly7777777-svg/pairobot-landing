Deployment Guide / 特務部署指南

Connect your bots with 3 simple steps. / 只需三個步驟，植入您的日常通訊軟體。

💬 LINE Bot

  1. Go to LINE Developers.
  2. Create a Messaging API channel.
  3. Get the Channel Access Token.
  4. Input your Webhook URL.
  5. 👉 Paste Token into PaiRobot App.

🏢 WeCom (企業微信)

  1. Log in to WeCom Admin.
  2. Create a new app in "App Management".
  3. Get CorpID, AgentId & Secret.
  4. Set up receiving messages with Token & AESKey.
  5. 👉 Paste all keys into PaiRobot App.

✈️ Telegram

  1. Search @BotFather on Telegram.
  2. Send /newbot command.
  3. Name your bot and get the HTTP API Token.
  4. 👉 Paste Token into PaiRobot App.
