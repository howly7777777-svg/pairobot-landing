Setup Guide

Connect your messaging agents in 3 steps.

LINE Bot

  1. Open LINE Developers Console.
  2. Create a Messaging API channel.
  3. Copy Channel Access Token.
  4. Paste it into PaiRobot Settings.

WeCom

  1. Log in to WeCom Admin.
  2. Create an application.
  3. Copy CorpID, AgentId, Secret.
  4. Paste all keys into PaiRobot Settings.

Telegram

  1. Search @BotFather.
  2. Send /newbot.
  3. Copy HTTP API Token.
  4. Paste it into PaiRobot Settings.
