Deployment Guide

Connect your bots with 3 simple steps.

💬 LINE Bot

  1. Go to LINE Developers.
  2. Create a Messaging API channel.
  3. Get the Channel Access Token.
  4. 👉 Paste Token into PaiRobot App.

🏢 WeCom

  1. Log in to WeCom Admin.
  2. Create a new app.
  3. Get CorpID, AgentId & Secret.
  4. 👉 Paste all keys into App.

✈️ Telegram

  1. Search @BotFather.
  2. Send /newbot command.
  3. Get the HTTP API Token.
  4. 👉 Paste Token into App.
