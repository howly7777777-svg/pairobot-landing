特務部署指南

只需三個步驟，將 PaiRobot 核心大腦植入您的日常通訊軟體。請選擇您想部署的平台：

💬

LINE 機器人

  1. 前往 LINE Developers 註冊開發者帳號。
  2. 建立一個 Messaging API 頻道。
  3. 在設定頁取得 Channel Access Token (長期)。
  4. 將您的 Webhook URL 填入並開啟 Use Webhook 功能。
  5. 👉 將 Token 填入 PaiRobot App 設定頁面。
🏢

企業微信 (WeCom)

  1. 登入 企業微信管理後台。
  2. 進入「應用管理」，點擊「建立應用」。
  3. 獲取您的 CorpID、AgentId 與 Secret。
  4. 在接收訊息設定中，配置 Token 與 EncodingAESKey。
  5. 👉 將上述五組金鑰填入 PaiRobot App 中。
✈️

Telegram

  1. 在 Telegram 搜尋 @BotFather。
  2. 輸入指令 /newbot 來建立一個新機器人。
  3. 為您的機器人命名，並取得 HTTP API Token。
  4. （選用）輸入 /setcommands 設定特務專屬選單。
  5. 👉 將 Bot Token 填入 PaiRobot App 即可連線。
